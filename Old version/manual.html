Introduction

Proto-layout is a software to draw solderless prototyping board layout. These layouts can be saved as *.pro which are editables or as as bitmap files. The bitmap file can be used to communicate the design to orders. For exemple a teacher can give those to his students to save time in laboratory as students won't have to figure out their own design. This is not a replacement for schematics design but a complement as students must learn to read electronic schematics as well as doing their own.

Proto-layout is so simple an obvious to use this manual is very brief.

Menus navigation

File menu

  • Prototyping board... This let you select the prototyping board size. Shortcut key: p
  • Open... This let you open a previously saved proto-layout ".pro" file for further editing. Shortcut key: o
  • Save... This let you save your proto-layout design as a ".pro" file that can be re-open for further editing. Shortcut key: s
  • Save as... Let you save your design under a new name.
  • Save to bitmap... Let you save your design as a bitmap file. That file can be edited in proto-layout
  • Quit Leave the application.

Edit

  • Undo Undo the last change. Shortcut key: <CTRL>-z
  • Redo Cancel the last undo. Shortcu key: <CTRL>-r

Jumper

Items in this menu let you choose jumper (wire) color. There is 6 predefined colors but you can also customize your own set of colors. The shortcut key: j display the color chooser dialog box.

Add

  • component... open the electronic components selector dialog. Components are grouped in categories. Components are represented as bitmap files, you can add component to database, this is explained below. Shortcut key: c
  • tag... Shortcut key: t. open the tag dialog box. This let you add text to identify component. After you have placed a tag over a component this tag follow the component, provided you drag the component by clicking over the tag. The font, size and color of tag are all selectable.

Help

  • manual... Open this manual in default web browser. Shorcut key: F1
  • about... display the about dialog box.

toolbar

The toolbar present the most used menu items as icons. Placing the mouse over an icon will display the corresponding tip. Each element of toolbar as a shorcut key as presented in above paragraph.

Placing elements on the board

When the mouse is moved over the board, its coordinates are displayed on the left of status bar. When the mouse is over a hole a small circle appear around that hole. When this append you can press down the left mouse button and drag the mouse over another hole and release it. This will place a jumper between the 2 holes. This jumper as the current selected color. You can change the color of an already placed jumper by right clicking on it and selecting color... of the popup context menu.

To place an electronic component click the corresponding icon on the toolbar or press c key to open component selection dialog box. Select the category then the component and click Ok button. The component will appear just under the toolbar on the left side. Drag the component on the board where you want to place it.

To add a tag click the tag icon on the toolbar or press t key to open the tag dialog box. The font button in the dialog open a font dialog. Tag, like components, appears under the toolbar at left and must be dragged on the prototyping board.

Elements are drawned on the board in the order they are created. If you add a component and place it over a tag, this one will become invisible. But there is an item in the popup context menu place it on top. Right click on the component and select place on top.

Adding component to the library

In the application directory there is a file named proto-layout.cmp. This is the component library file. This is a text file that can be open in any text editor. The file is formatted as an euphoria language object. It is a good idea to make a copy before editing it. If you don't understand, the format don't edit it. But if you understand it, you can add new board size, new component categories, and new components. The file is read a program startup. If you modify it while the program is running you must restart proto-layout for the update to take effect.

component bitmap

When you create a component bitmap. Don't use pure BLACK (RGB value 0x000000) as a color. Proto-layout use pure black as target points for prototyping board hole center. This is how it knows where to place jumper terminations. To fake pure black create a custom color with RGB value 0x010101. If you draw a new prototyping board use pure black only where there should be jumper termination.
Proto-layout expect bitmaps to be placed in resources sub-directory of the application directory.
